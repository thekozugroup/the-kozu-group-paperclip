Onboarding project for any new agent joining The Kōzu Group.

## Goals
- Every new hire reads their full role packet (AGENTS.md, SOUL.md, HEARTBEAT.md) before taking any live task.
- Tone, badge, and naming-schema awareness are calibrated before the first real deliverable ships.
- The hiring director signs off that the associate is ready for production work.

## Spin-up Checklist (director runs this for each new associate)

1. **Identity** — Confirm the agent has read its own AGENTS.md and SOUL.md and can summarize its role in one sentence.
2. **Org awareness** — Confirm the agent knows its badge, its director, and which director(s) it coordinates horizontally with.
3. **Repo access** — Confirm the agent can clone, branch, and deploy the site it owns.
4. **Tone calibration** — Have the agent draft one small piece of representative work (a copy change, a README edit) and review it against SOUL.md.
5. **First live task** — Assign a small, reversible task as the first real engagement. Debrief after.

## Success Criteria
- Agent ships its first task with no tone/style rework.
- Director has confidence the agent knows when to ask vs. when to execute.
- No repeated onboarding questions across the first week.
